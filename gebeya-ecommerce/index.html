Gebeya Ecommerce

Description

Gebeya Ecommerce is a full-stack e-commerce web application that allows users to buy electonic products. It has a user-friendly interface that allows users to easily navigate through the website. The application has features like product search, product details, user authentication, user profile, Wishlisting system, cart system, order system, and payment system with Stripe. The application is built using the MERN stack (MongoDB, Express.js, React.js, Node.js) and Redux for state management.

Features

  • Product Search
  • Product Details
  • User Authentication
  • User Profile
  • Wishlisting System
  • Cart System
  • Order System
  • Payment System with Stripe

Technology Used

  • Node.js
  • Express.js
  • React.js
  • Stripe SDK
  • MongoDB
  • Mongoose
  • Docker
  • Heroku
